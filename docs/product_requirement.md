# Product Requirement

## Project Name
Bookmark Manager

## Objective
Aplikasi web untuk menyimpan dan mengelola link pribadi berbasis akun.

## Core Features

1. Authentication
- user register
- user login
- user logout

2. Bookmark Management
- add bookmark
- edit bookmark
- delete bookmark
- view bookmark list

3. Bookmark Organization
- tag bookmark
- filter bookmark by tag

4. Search
- search bookmark by title
- search bookmark by url

## Tech Stack

Frontend
- React
- shadcn/ui

Backend
- Supabase

Database
- PostgreSQL

Hosting
- Vercel
